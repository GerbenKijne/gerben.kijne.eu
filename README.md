# gerben.kijne.eu

Personal site for Gerben Kijne — a single, self-contained static page.

- `index.html` — the entire site (HTML + CSS + a few lines of JS, no build step)
- Fonts loaded from Google Fonts; everything else is inline

## Local preview

```bash
python3 -m http.server 4599
# open http://localhost:4599
```

## Deploy

Hosted on Vercel, served at https://gerben.kijne.eu
